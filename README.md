# Partitions

## Using Partitions

The recommended way to use this crate is to add a line into your `Cargo.toml`
such as:

```toml
[dependencies]
partitions = "0.1"
```

and then add the following to to your `lib.rs` or `main.rs`:

```rust
extern crate partitions;
```
